To play the Context Game, give your AI agent the GitHub repo URL and say "I want to play the Context Game." Your agent will read the game protocol, set up your anonymous identity, and start matching your conversation against active questions.

When you ask a question naturally — "What's the best way to learn Python?" — your agent checks the game index. If a matching question exists, it shows you the current rankings and invites you to submit an entry or judge existing ones. If no question exists, you can create one.

All interactions happen through your normal chat interface. No website to visit, no account to create beyond GitHub. Your agent is your gateway to the game.
